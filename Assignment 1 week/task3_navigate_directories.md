# Task 3: Navigating Directories

## Description
Using the terminal, practice navigating through directories, listing file contents, and moving files to different locations.

## Instructions
1. **Navigate Directories**
    ```sh
    cd /path/to/directory
    ```

2. **List File Contents**
    ```sh
    ls -la
    ```

3. **Move Files**
    ```sh
    mv source_file /path/to/destination
    ```
