# Task 2: Execute Basic Linux Commands

## Description
Execute basic Linux commands (e.g., `ls`, `cd`, `mkdir`, `rm`, `touch`) to manipulate files and directories, with an emphasis on understanding their usage.

## Instructions
1. **List Files**
    ```sh
    ls
    ```

2. **Change Directory**
    ```sh
    cd directory_name
    ```

3. **Make Directory**
    ```sh
    mkdir new_directory
    ```

4. **Remove Directory**
    ```sh
    rm -r directory_name
    ```

5. **Create a File**
    ```sh
    touch newfile.txt
    ```
